# AI Work Simulator

An AI-powered virtual workplace simulation platform designed to help students, freshers, and professionals gain practical industry experience through interactive task simulations, AI-driven communication, and real-world workflow environments.

---

# Overview

AI Work Simulator is a full-stack application that recreates workplace scenarios using Artificial Intelligence. The platform allows users to experience professional environments, complete assigned tasks, interact with AI-powered modules, and improve workplace readiness through experiential learning.

This project was developed as part of a hackathon initiative focused on leveraging AI to bridge the gap between academic learning and industry expectations.

---

# Key Features

- AI-powered workplace simulation
- Interactive task and workflow management
- Modular frontend and backend architecture
- Dedicated AI services integration
- Real-time communication between modules
- Scalable and maintainable project structure
- User-friendly interface for simulation interaction
- Industry-oriented learning environment

---

# Problem Statement

Many students and fresh graduates lack practical workplace exposure before entering internships or full-time roles. Traditional learning methods often focus heavily on theory while providing limited hands-on industry experience.

AI Work Simulator addresses this challenge by providing:
- Simulated work environments
- AI-generated professional interactions
- Task-based learning experiences
- Practical exposure to workplace workflows

---

# Solution

The platform creates an adaptive virtual workplace where users can:
- Perform simulated professional tasks
- Interact with AI-driven systems
- Experience role-based workflows
- Improve communication and decision-making skills
- Prepare for real-world corporate environments

# System Architecture

The project follows a modular full-stack architecture for scalability and maintainability.

```text
Frontend (Client)
        │
        ▼
Backend API Server
        │
        ▼
AI Services Layer
        │
        ▼
Database / External Services
```

## Frontend
Handles:
- User interface
- User interaction
- Dashboard rendering
- API communication

## Backend
Responsible for:
- API management
- Business logic
- Request handling
- Authentication and routing

## AI Services
Manages:
- AI-generated responses
- Simulation logic
- Intelligent task interaction
- Dynamic workplace scenarios

# Installation

## Prerequisites

Ensure the following are installed on your system:

- Node.js
- npm or yarn
- Git

---

# Clone the Repository

```bash
git clone https://github.com/Divya-Jain07/AI_Work_Simulator.git
```

```bash
cd AI_Work_Simulator
```

---

# Install Dependencies

## Root Dependencies

```bash
npm install
```

## Frontend

```bash
cd client
npm install
```

## Backend

```bash
cd ../server
npm install
```

## AI Services

```bash
cd ../ai-services
npm install
```

---

# Running the Application

## Start Frontend

```bash
cd client
npm start
```

## Start Backend

```bash
cd server
npm start
```

## Start AI Services

```bash
cd ai-services
npm start
```
